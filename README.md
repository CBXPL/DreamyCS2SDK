# DreamyCS2SDK
DreamyCS2SDK is an optimized internal SDK for CS2.
Made with love by DreamyWare <3 developers for new developers to learn and understand latest CS2 structures!

## Avaible features in v0.1:
- 3D Box ESP (Self Explanatory)
- Interfaces: SigScan for ingame interfaces (more soon)
- Hooks: System for hooking ingame systems
- Tutorials: Built in tutorials how to modify each of the cs2 hooks
